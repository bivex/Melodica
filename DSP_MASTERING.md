# 🎛️ Руководство по DSPMasteringDesk

`DSPMasteringDesk` — это профессиональный инструмент для постобработки и финального мастеринга звука (Audio DSP) прямо внутри движка Melodica. Он опирается на библиотеку `pedalboard` (разработанную Spotify) для применения студийных эффектов к «сырым» стерео-массивам `numpy float32` или WAV-файлам.

Модуль позволяет превратить тихий, «сырой» микс из VST-синтезатора в плотный, коммерчески громкий и готовый к стримингам (Spotify, Apple Music) трек без клиппинга и искажений.

---

## 📦 Импорт

Модуль официально интегрирован в публичный API движка:

```python
from melodica import DSPMasteringDesk
from melodica import NewYorkCompressor  # Дополнительный модуль для параллельной компрессии
```

---

## 🚀 Быстрый старт: Мастеринг готового WAV-файла

Самый простой способ использовать деск — передать ему пути к входному и выходному файлу:

```python
from melodica import DSPMasteringDesk

# 1. Инициализируем мастеринг-деск
desk = DSPMasteringDesk(
    style="trap_drill",      # Жанровый пресет
    target_lufs=-10.0,       # Очень плотная громкость
    true_peak_ceiling=-1.0   # Безопасный потолок для стримингов
)

# 2. Обрабатываем аудиофайл
desk.master_file("input_raw_mix.wav", "output_mastered.wav")
```

## 🎚 Мастеринг аудио-массива (Numpy Float32)

Если вы работаете напрямую с рендером синтезатора (например, через `VSTPlayer`), используйте метод `process`:

```python
import numpy as np
from melodica import DSPMasteringDesk

# Предположим, у нас есть сырой стерео-массив аудио (2 канала)
# audio.shape -> (2, 441000)

desk = DSPMasteringDesk(style="pop_synthwave", target_lufs=-12.0)

# Мастерим сигнал (применяется EQ, компрессор, сатурация, LUFS гейн и лимитер)
mastered_audio = desk.process(audio)

# Теперь mastered_audio можно сохранить с помощью pedalboard.io.AudioFile или soundfile
```

## 🎧 Жанровые пресеты (`style`)

`DSPMasteringDesk` автоматически перестраивает цепь плагинов под конкретный стиль. Доступны 4 основных пресета:

1. **`pop_synthwave`** (По умолчанию)
   * Подчищает суб-низ (32 Гц).
   * SSL-компрессия шины (30мс атака, 150мс релиз) для склеивания микса.
   * Мягкий пленочный клиппинг (сатурация).
   * Tonal EQ (прибирает муть на 320 Гц, добавляет "воздух" на 10 кГц).
2. **`trap_drill`**
   * Сохраняет самый низкий суб-бас (срез от 28 Гц).
   * Медленная компрессия (35мс), пропускающая хлесткие транзиенты бочки.
   * Жесткий клиппинг для агрессивной гармонической накачки 808-х басов.
   * Буст 80 Гц (панч) и 12 кГц (сыпучий хэт).
3. **`ambient_classical`**
   * Максимально прозрачный тракт, сохраняющий оригинальную акустическую динамику.
   * Компрессор с ультранизким соотношением (ratio 1.2), никакого пампинга.
   * Кристально чистое расширение пространства (деликатный Reverb).
   * Отсутствие любых искажений (Distortion/Clipping выключены).
4. **`lofi`**
   * Винтажный звук: радио-срез снизу (45 Гц) и завал верхов (High Shelf cut).
   * Ламповый перегруз (Tube Distortion) для пыльного звучания.
   * Эффект старого сэмплера (12-bit Bitcrusher).
   * Агрессивно качающая компрессия (быстрая атака 15мс).

## 🔊 Управление громкостью (LUFS & True Peak)

Алгоритм встроенной компенсации громкости основан на высокоточном K-weighted RMS, который математически проецируется в стандарт **LUFS**.

* **`target_lufs`**: 
  * `-14.0`: Золотой стандарт стримингов (Spotify / Apple Music). Идеальный баланс плотности и динамики.
  * `-12.0`: Современный коммерческий поп/электроника.
  * `-9.0` до `-10.0`: Ультрагромкий клубный Phonk, Drill, EDM.
* **`true_peak_ceiling`**:
  * Финальный жесткий лимитер. Оставьте `-1.0 dBFS`, чтобы избежать межсэмпловых пиков (ISP) при перекодировании в MP3/AAC.

## 🗽 Дополнительно: NewYorkCompressor

Для работы с отдельными дорожками (особенно барабанами) в Melodica встроен алгоритм классической параллельной компрессии:

```python
from melodica import NewYorkCompressor

ny_comp = NewYorkCompressor(
    dry_mix=0.75,     # 75% исходного панча
    wet_mix=0.6,      # 60% расплющенного сигнала (сустейн и плотность)
    threshold_db=-24.0,
    ratio=10.0,
    attack_ms=2.0
)

fat_drums = ny_comp.process(raw_drums_array)
```

## ⚠️ Graceful Fallback (Безопасный режим)

Если библиотека `pedalboard` не установлена, система не упадет с ошибкой! 
Она перейдет в режим высокоточной математической нормализации (пиковый безопасный фоллбэк), а для параллельного компрессора применит аналоговую Soft-Knee Tanh математику.
